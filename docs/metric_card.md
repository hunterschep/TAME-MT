# Metric Card: TAME-MT Report

## Metric Name

TAME-MT report.

## Primary Metrics

- TM-BLEU and TM-chrF
- delta over TM for BLEU and chrF
- SourceExposure
- PairLeak at configured thresholds
- Far-BLEU and Far-chrF
- GenGap-BLEU and GenGap-chrF

## Intended Use

TAME-MT is intended to contextualize machine translation quality scores with
train-test proximity. It is especially useful for low-resource MT, narrow-domain
benchmarks, benchmark audits, and paper reports that need to distinguish
in-domain performance from broader generalization evidence.

## Primary Audience

MT researchers, low-resource MT researchers, benchmark builders, reviewers, and
evaluation pipeline maintainers.

## Not Intended For

TAME-MT is not intended to replace human evaluation, BLEU, chrF, or learned
semantic metrics. It does not prove memorization, evaluate without training
data, or decide whether a benchmark is valid.

## Inputs Required

Canonical scoring mode requires aligned UTF-8 text files for training source,
training target, test source, test reference, and system hypothesis.

Audit mode can run without a hypothesis file. Source-only audit is supported,
but target, pair, TM, and delta metrics require the corresponding target or
reference data.

## Method Summary

TAME-MT normalizes text with NFKC and whitespace collapse, extracts character
3-5-gram sets, and computes Jaccard similarity. SourceExposure is the nearest
training-source similarity for each test source. TM hypotheses are generated by
retrieving the closest training source and outputting its paired target.

Pair exposure is computed by reranking the union of source and target top-k
candidate training indices. Exact pair overlap is computed exactly.

The default translation-memory zero policy outputs an empty string when a test
source shares no character n-grams with any training source. This avoids giving
the TM baseline accidental credit from an arbitrary training target.

Default index mode is `auto`: native Rust exact retrieval for small corpora and
native Rust fast rare-gram candidate retrieval for larger corpora when the
extension is installed. Source installs without the extension fall back to the
same exact/fast pure-Python modes. The signature records the requested mode,
resolved backend, and fast-mode bounds.

## Known Limitations

The default similarity is surface-based and may miss semantic paraphrases.
Thresholds are interpretable defaults, not universal constants. Pair exposure
uses an approximation in v0.1. BLEU on small bins can be unstable.

TAME-MT should be read as a report, not as a single scalar "TAME score."

Fast retrieval mode is approximate for nearest-neighbor exposure. Exact overlap
statistics remain exact.

## Privacy And Safety

Training corpora may contain sensitive or community-owned text. TAME-MT does
not print neighbor training text by default and does not send data anywhere.
Segment-level text fields are opt-in.
